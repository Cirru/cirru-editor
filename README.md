
Cirru Editor
----

built with React

### Keyboard

```cirru
tab
  token packNode
  expr packNode

shift tab
  token --
  expr unpackExpr

enter
  token afterToken
  expr afterToken

shift enter
  token beforeToken
  expr beforeToken

command enter
  token --
  expr prependToken
```

### Focus

```cirru
focus-to
focus-forward
focus-backward
focus-inside
focus-outside
```
